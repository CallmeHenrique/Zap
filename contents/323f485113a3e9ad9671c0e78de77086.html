Cadastrar Fotógrafo

UMLUseCase
Empresa ZAP :: Casos de Uso :: Cadastrar Fotógrafo

Description

Fluxo Principal: 1. Fotógrafo acessa aba de perfil do usuário; 2. Sistema exibe informações sobre o fotógrafo; 3. Fotógrafo clica em cadastro; 4. Sistema exibe campos de cadastro; 5. Fotógrafo preenche os campos de cadastro de usuário e dos equipamentos; 6. Sistema valida as informações inseridas [FS001]; 7. Sistema retorna a mensagem "Cadastro realizado com sucesso". Fluxo Secundário: FS001: 6.1. Sistema valida as informações e encontra uma inconsistência nos dados inseridos; 6.2. Sistema exibe uma mensagem de erro para o fotógrafo dizendo que as informações estão inconsistentes; 6.3. Sistema retorna para o fluxo principal, passo 4.
