Realizar Pagamento

UMLUseCase
Empresa ZAP :: Casos de Uso :: Realizar Pagamento

Description

Fluxo Principal: 1. Cliente verifica o valor a ser pago; 2. Cliente acessa o aplicativo do banco; 3. Cliente realiza o pagamento no PIX do Fotógrafo; 4. Cliente abre o aplicativo ZAP; 5. Cliente abre a sessão; 6. Sistema mostra informações da sessão 7. Cliente envia comprovante de pagamento; 8. Sistema confirma o valor pago.
