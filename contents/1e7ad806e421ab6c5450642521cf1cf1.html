Cadastrar Usuário

UMLUseCase
Empresa ZAP :: Casos de Uso :: Cadastrar Usuário

Description

Fluxo Principal: 1. Cliente acessa aba de perfil do usuário; 2. Sistema exibe informações sobre o cliente; 3. Cliente clica em cadastro; 4. Sistema exibe campos de cadastro; 5. Cliente preenche os campos de cadastro de usuário; 6. Sistema valida as informações inseridas [FS001]; 7. Sistema retorna a mensagem "Cadastro realizado com sucesso". Fluxo Secundário: FS001: 6.1. Sistema valida as informações e encontra uma inconsistência nos dados inseridos; 6.2. Sistema exibe uma mensagem de erro para o cliente dizendo que as informações estão inconsistentes; 6.3. Sistema retorna para o fluxo principal, passo 4.
