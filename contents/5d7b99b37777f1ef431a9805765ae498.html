Agendar Sessão

UMLUseCase
Empresa ZAP :: Casos de Uso :: Agendar Sessão

Description

Fluxo Principal: 1. Cliente seleciona o agendamento de sessão na página inicial; 2. Cliente preenche informações a respeito de como deseja que sua sessão aconteça; 3. Sistema faz a validação dos dados preenchidos pelo cliente [FS001]; 4. Sistema apresenta sugestões com base na sessão descrita pelo cliente; 5. Cliente escolhe com quem deseja fazer a sessão de fotos; 6. Sistema exibe os dados preenchidos pelo cliente; 7. Cliente confirma os dados de agendamento; 8. Sistema exibe mensagem de sucesso, e informa que o ator deve aguardar. Fluxo Secundário: FS001: 3.1. Sistema verifica incoerência nos dados preenchidos. 3.2. Sistema informa a inconsistência dos dados 3.3. Sistema realiza retorno ao passo 2 do fluxo principal
